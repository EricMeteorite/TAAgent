# HDRI 拍摄与处理指南

## 一、拍摄准备

### 设备要求

| 设备 | 要求 | 说明 |
|------|------|------|
| 相机 | 单反/无反 | 支持手动模式、RAW 格式 |
| 镜头 | 鱼眼镜头 | 180° 视角，覆盖半球 |
| 三脚架 | 稳固 | 包围曝光必须 |
| 灰卡 | 18% 灰 | 白平衡参考 |
| 色卡 | X-Rite 等 | 色彩校正参考 |

### 相机设置

```
格式: RAW
白平衡: 6500K (固定！不要自动)
ISO: 最低原生值 (通常 100-200)
光圈: f/8 - f/11 (最佳画质)
对焦: 手动对焦，设置在无穷远
```

## 二、包围曝光拍摄

### 参数配置

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| 张数 | 7-9 张 | 覆盖全动态范围 |
| EV 间隔 | ±2 EV | 每档 2 档间隔 |
| 动态范围 | 20+ EV | 有太阳场景必须 |

### 拍摄流程

```
1. 放置灰卡/色卡（可选，作 Ground Truth）
2. 设置相机: ISO 100, f/8, 6500K
3. 测光确定中间曝光值
4. 包围曝光拍摄:
   - 最暗: EV-6 到 EV-8 (太阳区域不能过曝)
   - 最亮: EV+6 到 EV+8 (暗部有细节)
```

### 检查点

**关键检查**: 最暗那张照片中，太阳位置不能是纯白！

```
检查方法:
1. 在相机上查看直方图
2. 最暗照片的高光区域不能有溢出
3. 如果太阳是纯白 → 继续降低曝光

判断标准:
✓ 最暗照片: 太阳区域有颜色信息，不是纯白
✓ 最亮照片: 暗部接近中灰
✓ 中间照片: 整体曝光正常
```

## 三、后期处理

### HDRI 合成

```
软件: Photoshop / Photomatix / PTGui

步骤:
1. 导入所有曝光照片
2. 对齐图像（如有微小位移）
3. Merge to HDR 32-bit
4. 导出为 .hdr 或 .exr 格式
```

### 色彩校正

```
参考灰卡/色卡:
1. 使用灰卡设置中性灰
2. 检查色卡各色块是否准确
3. 避免大幅调整色温
```

### 太阳处理

```
选项 A: 保留太阳
- 确保动态范围足够 (20+ EV)
- 太阳位置有正确的亮度信息

选项 B: 移除太阳
- 后期去除太阳区域
- 在 UE 中用 DirectionalLight 代替
- 优点: 色温更准确
```

### 分辨率与比例

```
推荐分辨率: 8192 × 4096 或更高
宽高比: 2:1 (球形映射)

UE 导入后:
- Maximum Cubemap Size: 根据需要设置
- 通常 4096 或 2048 足够
```

## 四、UE 导入与使用

### 导入设置

```
Import Settings:
- sRGB: False (重要！)
- Compression: HDR (RGB, no sRGB)

纹理设置:
- Mip Gen Settings: LeaveExistingMips
- Max Texture Size: 根据需要
```

### SkyLight 配置

```
SkyLight Component:
- Source Type: SLS_SpecifiedCubemap
- Cubemap: [导入的 HDRI]
- Intensity: 1.0 (或根据 EV 计算)

如果使用 EV = 13:
  Emissive Intensity = 8192 (2^13)
```

### 天空球材质

```
Material:
- Shading Model: Unlit
- Emissive Color: HDRI 贴图 × EV倍率

材质属性:
- Emissive: HDRI_Texture
- 乘数: pow(2, EV) = 8192 (对于 EV=13)
```

## 五、常见问题

### Q: HDRI 导入后颜色不对？

检查:
1. sRGB 是否关闭
2. 白平衡是否固定 6500K
3. 后期是否调整了色温

### Q: 天空球亮度不够？

```
解决方案:
1. 检查 Emissive Intensity 设置
2. 确保材质是 Unlit
3. 计算: Intensity = 2^EV
```

### Q: 太阳位置不对？

```
使用 Sun Position Calculator:
1. 记录拍摄时的时间、地点
2. 在 UE 中设置相同参数
3. 太阳方向自动匹配 HDRI
```

### Q: HDRI 动态范围不够？

```
症状:
- 太阳区域过曝
- 暗部丢失细节
- 照明不够真实

解决方案:
1. 拍摄更多曝光档位
2. 增加包围曝光范围
3. 确保 EV 差距足够大
```

## 六、Ground Truth 拍摄

### 辅助道具

| 道具 | 用途 | 放置位置 |
|------|------|----------|
| 灰球 | 漫反射质感参考 | 场景中心 |
| 金属球 | 反射参考 | 场景中心 |
| 灰卡 | 色温曝光参考 | 画面边缘 |
| 色卡 | 综合色彩参考 | 画面边缘 |
| 灰阶 | Gamma 参考 | 画面边缘 |

### 拍摄要求

```
相机设置: 与 HDRI 拍摄相同
位置: 在 HDRI 拍摄点附近
角度: 包含所有辅助道具
格式: RAW + 包围曝光
```

### 后期匹配

```
对比方法:
1. 在 UE 中放置相同的辅助道具模型
2. 使用相同的相机设置渲染
3. 与照片并排对比
4. 调整灯光参数直至匹配

关键对比点:
- 灰球: 高光、阴影过渡
- 金属球: 环境反射
- 灰卡: 中灰亮度
- 色卡: 各色块色彩
```

## 七、Virtual Production HDRI

### LED Wall 场景

```
拍摄要点:
1. LED 屏幕本身作为光源
2. 拍摄 LED 屏显示内容
3. 测量实际照度
4. 记录色温

UE 匹配:
1. LED 背景作为 HDRI
2. 根据测量照度调整强度
3. 色温与现场同步
```

### 实拍 + 虚拟合成

```
同步拍摄:
1. 现场拍摄 Ground Truth
2. 同步记录 HDRI
3. 记录相机参数 (EV, 白平衡, 镜头)

UE 匹配:
1. 使用相同 EV 值
2. 相机白平衡同步
3. 灯光方向与照片一致
```
